# Clock

A lightweight clock application built with pure HTML and CSS.

## Tech Stack
- HTML
- CSS

## Getting Started
```bash
git clone https://github.com/mrsandeep27/clock.git
cd clock
```

Open `index.html` in your browser — no build step needed.

## Features
- Clean visual clock display
- Zero JavaScript dependencies
- Instant loading, no frameworks
